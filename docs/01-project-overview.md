# 01. 프로젝트 개요

## 서비스 정보

| 항목 | 내용 |
|------|------|
| **서비스명** | 글리움 (Gleaum) |
| **슬로건** | 반짝이는 글이 모여있는 공간 / 가족의 모든 일정을 한 곳에서 |
| **플랫폼** | iOS / Android / Web (PWA) |
| **타겟** | 자녀를 둔 3~5인 가족 (부모 + 자녀) |
| **런타임** | 실서비스 목표 |

---

## 핵심 가치

글리움은 **가족 단위**의 일정 관리 앱입니다. 단순한 캘린더가 아니라:

- 부모가 자녀 일정을 등록하고, 자녀가 완료 처리하면 부모가 실시간으로 확인
- 가족 전체가 공유하는 일정과 개인 일정을 한 화면에서 관리
- 교육비·공과금 등 정기지출을 자동 가계부로 정리
- 자녀 일정 미완료 시 부모에게 재알림

---

## 일정 유형 4가지

| 유형 | 설명 | 컬러 |
|------|------|------|
| **공유일정** (shared) | 가족 전체가 함께하는 일정 (가족여행, 외식 등) | `#5A32FA` 보라 |
| **개인일정** (personal) | 개인 일정 (치과, 미용실 등) | `#06B6D4` 청록 |
| **자녀일정** (child) | 자녀 학원·활동 일정 (부모가 등록, 완료 확인) | `#10B981` 에메랄드 |
| **정기지출** (expense) | 매월 반복되는 지출 (학원비, 공과금, 구독료 등) | `#F59E0B` 앰버 |

---

## 자녀 일정 상태 흐름

```
대기중(pending) → 진행중(in_progress) → 완료(completed)
                                       → 미완료(missed)
```

- **부모**가 자녀 일정 등록
- 일정 시작 시각이 되면 자동으로 `in_progress`
- **자녀** 또는 **부모**가 완료/미완료 처리
- 미완료 시 부모에게 재알림 발송

## 사용자 역할

| 역할 | 설명 |
|------|------|
| `parent` | 부모 — 가족 그룹 생성/관리, 모든 일정 접근 |
| `child` | 자녀 — 본인 일정 확인 및 완료 처리 |
| `guest` | 미사용 (향후 확장용) |
