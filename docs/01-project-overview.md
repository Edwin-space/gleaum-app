# 01. 프로젝트 개요

## 서비스 정보

| 항목 | 내용 |
|------|------|
| **서비스명** | 글리움 (gleaum) |
| **슬로건** | 나의 일상부터, 우리가 연결되는 순간까지 / 우리의 시차를 없애는 가장 스마트한 방법 |
| **플랫폼** | iOS / Android / Web (PWA) |
| **타겟** | 1인 가구 (스마트한 일정 관리), 연인 (서로의 일상 공유), 가족 (육아/생활 동기화), 지인 (1회성 모임 초대) |
| **런타임** | 실서비스 목표 |

---

## 핵심 가치

글리움은 단순한 저장소가 아닌 **유연하고 확장 가능한 일상 네트워크 플랫폼**입니다.

- **Visibility (비춤)**: 복잡한 일상을 선명하게 시각화하여 "묻지 않아도 아는 배려" 제공
- **Commitment (약속)**: 함께할 시간을 정성스럽게 기록하고, 지인/친구에게 1회성 초대 링크로 간편하게 공유
- **Synchronized Space (동기화 공간)**: 나, 연인, 가족의 일정이 시차 없이 하나로 연결되어 머무는 디지털 광장
- (예정) **OS Native Calendar 연동**: 스마트폰 기본 캘린더와 완벽한 양방향 동기화

## 일정 유형 4가지

| 유형 | 설명 | 테마 |
|------|------|------|
| **공유일정** (shared) | 연인, 가족, 또는 특정 그룹이 함께하는 일정 / 1회성 지인 초대 모임 | Brand Blue (`#0084CC`) |
| **개인일정** (personal) | 나만의 개인 일정 (치과, 미용실 등) | Brand Teal (`#0CC9B5`) |
| **자녀/그룹일정** (child) | 자녀 학원·활동, 또는 그룹의 특정 목표 일정 (진척도 확인용) | Brand Green (`#2EE895`) |
| **정기지출** (expense) | 매월 반복되는 지출 (학원비, 공과금, 구독료 등) | Accent Amber |

---

## 그룹/자녀 일정 상태 흐름

```
대기중(pending) → 진행중(in_progress) → 완료(completed)
                                       → 미완료(missed)
```

- 일정 등록 후 시간에 따라 자동 상태 전이
- 완료/미완료에 따른 실시간 푸시 알림으로 커뮤니케이션 오차 제로화

---

## 사용자 역할

| 역할 | 설명 |
|------|------|
| `parent` / `admin` | 기본 사용자 — 그룹 생성/관리, 일정 등록 및 관리, 지인 초대 링크 발급 |
| `child` / `member` | 종속 사용자 — 본인 할당 일정 확인 및 상태(완료) 처리 |
| `guest` | 1회성 초대 링크로 접속하여 특정 일정만 확인하는 외부 사용자 |
