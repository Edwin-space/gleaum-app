# 💰 지출 카테고리 (Expenses)

지출은 성격에 따라 **고정지출(매달 통제하기 어려운 고정 비용)**과 **변동지출(유동적으로 통제 가능한 비용)**로 구분하여 설계되었습니다. 기본적으로 **1차(대분류) - 2차(중분류)** 구조를 기본 데이터셋으로 제공하며, **3차(소분류)**는 유저가 직접 추가하거나 선택할 수 있는 가이드라인 역할을 합니다.

---

## 🏠 1. 고정 지출 (Fixed Expenses)

### 1.1 주거 / 통신
* **2차 카테고리:** 월세 / 대출상환, 관리비, 공과금 (전기/수도/가스), 통신비 (휴대폰/인터넷/TV)
* *3차 카테고리 추천 (사용자 정의):* 버팀목대출, 아파트관리비, 도시가스, OTT 결합 상품 등

### 1.2 보험 / 세금
* **2차 카테고리:** 보장성 보험, 연금 보험, 정기 세금 (재산세/자동차세)
* *3차 카테고리 추천 (사용자 정의):* 실비보험, 암보험, 자동차세, 주민세 등

### 1.3 구독 / 정기결제
* **2차 카테고리:** OTT 서비스, 소프트웨어 / 앱 멤버십, 정기 배송
* *3차 카테고리 추천 (사용자 정의):* 넷플릭스, 쿠팡 와우, 유튜브 프리미엄, 클라우드 스토리지 등

### 1.4 교육 / 육아
* **2차 카테고리:** 학원비 / 등록금, 아이 용품, 용돈 (자녀)
* *3차 카테고리 추천 (사용자 정의):* 영어학원, 유치원 원비, 기저귀/분유, 학습지 등

---

## 🛒 2. 변동 지출 (Variable Expenses)

### 2.1 식비
* **2차 카테고리:** 식재료 (장보기), 외식, 배달 음식, 카페 / 디저트
* *3차 카테고리 추천 (사용자 정의):* 대형마트, 마켓컬리, 배달의민족, 스타벅스 등

### 2.2 생활 / 마트
* **2차 카테고리:** 생필품, 가구 / 가전, 반려동물, 뷰티 / 미용
* *3차 카테고리 추천 (사용자 정의):* 다이소, 미용실, 올리브영, 사료/간식 등

### 2.3 패션 / 잡화
* **2차 카테고리:** 의류, 신발 / 가방, 액세서리
* *3차 카테고리 추천 (사용자 정의):* SPA 브랜드, 백화점 쇼핑, 온라인 쇼핑몰 등

### 2.4 교통 / 차량
* **2차 카테고리:** 대중교통, 택시, 주유 / 충전, 차량 정비 / 주차
* *3차 카테고리 추천 (사용자 정의):* 하이패스, 엔진오일 교환, 세차비, 대리운전 등

### 2.5 문화 / 여가
* **2차 카테고리:** 영화 / 공연 / 전시, 도서 / 음반, 여행 / 숙박, 취미 / 운동
* *3차 카테고리 추천 (사용자 정의):* 헬스장 등록, 캠핑 용품, 숙박 앱 결제, 영화관 등

### 2.6 의료 / 건강
* **2차 카테고리:** 병원비, 약국, 영양제 / 건강기능식품
* *3차 카테고리 추천 (사용자 정의):* 치과, 한의원, 유산균/비타민 등

### 2.7 경조사 / 선물
* **2차 카테고리:** 경조사비 (축의/조의), 기념일 선물, 모임 회비
* *3차 카테고리 추천 (사용자 정의):* 부모님 용돈, 친구 생일 선물, 동창회비 등
