
### 기본 카테고리

```text

```

선정 이유: 글리움의 1차 사용 목적은 일정·캘린더·생활 흐름 관리이며, 가계부는 이를 보완하는 개인 관리 기능이다. 금융 서비스로 오인될 수 있는 `금융`보다 `생산성`이 현재 제품과 정확히 일치한다.

### 권장 태그 — 콘솔에서 제공되는 항목 중 최대 5개

우선순위:

1. 캘린더
2. 할 일 목록
3. 개인 금융
4. 가족
5. 라이프스타일

태그 명칭은 Play Console에서 실제 제공되는 한국어 항목을 우선한다. 해당 태그가 없으면 가장 가까운 제안 태그만 선택하고, 관련성이 낮은 태그를 검색 노출 목적으로 추가하지 않는다.

### 자연스럽게 반영한 검색어

```text
일정 관리, 공유 일정, 캘린더, 가계부, 수입 지출, 정기 지출,
가족 일정, 커플 일정, 친구 모임, 생활 관리
```

별도의 키워드 입력란은 없으므로 앱 이름·간단한 설명·자세한 설명에서 문맥에 맞게 사용한다. 같은 단어를 반복해서 나열하지 않는다.

## 6. Play Console 인접 설정 권장값

### 기본 언어

```text
한국어(대한민국) — ko-KR
```

현재 앱 UI와 고객지원이 한국어 중심이므로 앱 전체 영문 현지화 전에는 영문 스토어 등록정보를 별도로 운영하지 않는다.

### 광고 포함

```text
예
```

Android 앱에 광고 SDK와 광고 지면이 포함되어 있으므로 실제 노출 빈도와 관계없이 `광고 포함`으로 선언한다.

### 대상 연령

```text
현재 운영 기준: 만 18세 이상
```

가족 공간의 자녀 계정·보호자 동의·연령별 기능 제한이 운영 검증되기 전에는 아동 대상 앱으로 홍보하지 않는다. 해당 기능을 정식 공개할 때 대상 연령, 가족 정책, 광고 SDK 적합성을 다시 심사한다.

### 앱 액세스

```text
일부 또는 모든 기능이 제한됨 — 로그인 필요
```

심사용 계정의 이메일과 비밀번호는 Play Console의 앱 액세스 항목에 별도로 입력한다. 개인 Google 계정이나 실제 사용자 계정을 심사 정보로 사용하지 않는다.

### 개인정보 및 데이터 보안

데이터 보안 답변은 등록정보 카피가 아니라 실제 앱·서버·SDK 동작을 기준으로 별도 검증한다. 최소 검토 대상은 다음과 같다.

- 계정 정보: 이메일, 사용자 ID, 표시 이름
- 사용자 콘텐츠: 일정, 공간 소식, 가계부 입력 내용
- 앱 활동: 화면 접근, 기능 사용 이벤트
- 앱 정보 및 성능: Crashlytics 충돌 로그, 진단 정보
- 기기 또는 기타 ID: FCM 토큰, 광고·분석 SDK 식별자
- Android 기기 캘린더: 사용자가 권한을 허용한 경우에만 처리
- 광고 SDK를 통한 데이터 처리와 제3자 공유 여부

위 항목은 최종 AAB에 포함된 SDK와 개인정보처리방침을 대조한 뒤 확정한다.

## 7. 사용하지 않을 표현

다음 표현은 기능 범위 또는 Google Play 정책상 사용하지 않는다.

- `최고`, `1위`, `필수 앱`, `완벽한`, `유일한`
- `무료`, `지금 다운로드`, `설치하세요`, 기간 한정 혜택
- 아직 운영 검증되지 않은 자녀 위치 확인·자동 GPS 알림
- 아직 지원하지 않는 iOS·macOS 동기화
- 구현되지 않은 파일·사진 첨부 또는 완전한 양방향 캘린더 동기화
- 공유 공간 지출이 개인 가계부에 자동 반영된다는 표현
